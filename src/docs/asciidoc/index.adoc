== Error Response Policy

SmartBiz ERP의 모든 REST API는 오류 발생 시 아래의 **공통 응답 형식**을 따른다.

[source,json]
----
{
  "success": false,
  "data": null,
  "message": "에러 메시지"
}
----

* `success` : 요청 성공 여부 (항상 false)
* `data` : 오류 응답에서는 항상 null
* `message` : 오류 원인에 대한 설명 메시지

=== HTTP Status Codes

|===
| Status | Name | 설명

| 400 | Bad Request | 요청 파라미터 오류 또는 유효성 검증 실패
| 404 | Not Found | 요청한 리소스를 찾을 수 없음
| 409 | Conflict | 비즈니스 로직 충돌
| 500 | Internal Server Error | 서버 내부 오류
|===

=== Error Examples

==== 400 Bad Request
[source,json]
----
